Quick start options:

- Clone the repo: `git clone https://github.com/up-pro/my-portfolio.git`.

## Installation

```
$ git clone git@github.com:patrikrobredo/my-portfolio.git
$ cd my-portfolio
$ npm install
$ npm start
```
